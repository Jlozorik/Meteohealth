# 2.3 Пользовательский интерфейс

## Технологии

UI полностью построен на **Jetpack Compose + Material 3**. Навигация — `Navigation Compose` с `ModalNavigationDrawer`. `ExperimentalMaterial3ExpressiveApi` подключён глобально через `freeCompilerArgs`.

## Тема

Только светлая тема, `dynamicColor = false`. Тёмная тема не реализована.

| Роль | Цвет | Hex |
|---|---|---|
| Background | Крем | #F7F4EE |
| Primary | Лесной | #1F4E3D |
| Secondary (акцент) | Терракота | #B85C38 |
| Tertiary | Серо-синий | #4D5A6A |
| RiskCalm | Зелёный | #3D8B40 |
| RiskWatch | Янтарный | #C9A227 |
| RiskAlert | Оранжевый | #D17A39 |
| RiskHigh | Красный | #B23A3A |

## Типографика

| Стиль | Семейство | Особенности |
|---|---|---|
| displayLarge | FontFamily.Monospace | индексы, цифры |
| headlineMedium | FontFamily.Monospace | заголовки экранов |
| labelLarge | FontFamily.SansSerif | letter-spacing = 6sp, для секционных заголовков |
| NumericStyle | Monospace 72sp | fontFeatureSettings="tnum", выравнивание цифр |

## Навигация

`ModalNavigationDrawer` + `NavHost`. Боковое меню (`MeteoDrawer`) содержит 4 пункта и версию приложения в футере.

| Маршрут | Экран |
|---|---|
| onboarding | OnboardingScreen |
| home | HomeScreen |
| forecast | ForecastScreen |
| journal | JournalScreen |
| settings | SettingsScreen |

Нижней панели навигации нет.

## Компоненты

- **MeteoDrawer** — `ModalDrawerSheet`, 4 `NavigationDrawerItem`, футер с версией.
- **MeteoTopBar** — `CenterAlignedTopAppBar` с иконкой меню-гамбургера.
- **DividedSection** — `Column` с `labelLarge`-заголовком и `HorizontalDivider`-ами.
- **EmptyState** — пустое состояние с иконкой и текстом.

## Экраны

### Онбординг (OnboardingScreen)

4 шага с анимацией перехода:

| Шаг | Содержимое |
|---|---|
| 0 | Имя, возраст (опционально) |
| 1 | Чувствительность (Slider 1–5) |
| 2 | Хронические состояния (Chip-группа) |
| 3 | Город (TextField + кнопка «Определить автоматически») |

Кнопки «Далее» / «Назад» / «Готово». Заголовок каждого шага — телеграфный стиль («Кратко о себе.», «Насколько чувствуешь.»).

### Главный экран (HomeScreen)

Секции через `DividedSection`:

- **Индекс** — крупное число (NumericStyle), вердикт («Спокойно.», «Береги голову.» и т.д.), цвет по `RiskLevel`.
- **Погода** — температура, давление, влажность, ветер.
- **Магниты** — Kp-индекс, текстовая интерпретация.
- **Что сделать** — рекомендации из `RecommendationGateway` по текущему `RiskLevel`.

Кнопка обновления в TopBar запускает `RefreshNowUseCase`.

### Прогноз (ForecastScreen)

Список `DayRow` на 5 дней. Каждая строка: дата, диапазон температур, Kp, RiskLevel (цветной текст). Раскрывающийся элемент (ToggleDay) показывает почасовой график давления (Vico).

Переключатель единиц давления (гПа / мм рт.ст.) в TopBar.

### Журнал (JournalScreen)

Вкладки «Записи» / «Триггеры» (`TabRow`).

**Записи** — список `JournalEntry` в обратном порядке. FAB-like кнопка «Отметить» открывает bottom sheet с полями: уровень самочувствия (Slider), симптомы (Chip), заметки (TextField).

**Триггеры** — результаты `PearsonAnalyzer` по трём факторам (давление, температура, Kp). При < 14 записей — «Мало записей. N из 14.»

### Настройки (SettingsScreen)

Аккордеон секций (`ExpandableSection`):

- **Уведомления** — тумблер + типы событий.
- **Единицы** — гПа / мм рт.ст.
- **Город** — текущий город + кнопка «Изменить».
- **Откуда данные** — OWM + NOAA SWPC.
- **Приватность** — локальное хранение, без передачи третьим лицам.
- **Данные** — «Стереть всё» с `AlertDialog`-подтверждением.
- **О приложении** — версия.

## Стиль копирайтинга

Все UI-тексты — телеграфный стиль: короткие предложения с точкой в конце.

| Было (многословно) | Стало |
|---|---|
| Нет записей в журнале | Дневник пуст. |
| Состояние неизвестно | Нет данных. |
| Что-то пошло не так, попробуйте ещё раз | Что-то пошло не так. |
| Следите за своим давлением сегодня | Следи за давлением. |

## Иконка приложения

Adaptive icon: фон `#F7F4EE` (крем), foreground — две горизонтальные полосы `#1F4E3D` + терракотовая точка `#B85C38` справа.
