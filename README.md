# Метео-Здоровье

Android-приложение для людей, чьё самочувствие зависит от погоды. Отслеживает атмосферное давление, температуру и геомагнитную активность, считает персональный индекс самочувствия и предупреждает о неблагоприятных условиях заранее.

## Что умеет

- Рассчитывает индекс самочувствия (0–100) по погодным данным и профилю пользователя
- Показывает прогноз на 5 дней с графиками температуры
- Ведёт дневник самочувствия и анализирует личные триггеры через корреляцию Пирсона
- Обучает линейную регрессию на записях дневника и выдаёт персональный прогноз
- Уведомляет в фоне, когда условия ухудшаются

## Стек

Kotlin 2.2 · Jetpack Compose · Material 3 · Room · Retrofit · Koin · WorkManager · Vico

Данные погоды: OpenWeatherMap. Геомагнитная активность: NOAA SWPC (без ключа).

## Сборка

Добавьте ключ OpenWeatherMap в `local.properties`:

```
OPEN_WEATHER_API_KEY=ваш_ключ
```

Без ключа приложение запускается в demo-режиме с синтетическими данными.

```bash
./gradlew assembleDebug
```
